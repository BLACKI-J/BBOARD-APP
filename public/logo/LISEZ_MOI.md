# Changer le logo de l'application BBOARD

Ce dossier est l'emplacement officiel pour le logo de votre structure (association, mairie, colonie...).

Pour personnaliser le logo dans toute l'application (Menu principal, Impressions PDF des formulaires Cerfa, etc.) :

1. Prenez l'image de votre propre logo (idéalement avec un fond transparent, format PNG).
2. Nommez ce fichier EXACTEMENT `logo.png`.
3. Écrasez le fichier existant ou placez-le dans ce dossier (le chemin devra être `public/logo/logo.png`).
4. C'est tout ! Rechargez la page web de l'application (ou videz le cache de votre navigateur `Ctrl+F5`) et votre logo apparaîtra automatiquement partout.

*Astuce : Si vous ne mettez aucun fichier `logo.png` ici, l'application utilisera le logo générique (éclair) par défaut dans le menu.*
